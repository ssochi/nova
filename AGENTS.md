## Define the Project's Core Goal (what to do, how to measure it)
从0到1使用rust制作一个golang编译器,要求:
1. 只能使用标准库.
2. 先实现VM在实现完整后端编译器
3. 需要支持最新golang所有特性
4. 支持所有内置函数和标准库.
5. 目标是可以编译一个复杂的golang开源项目,并正常运行.
6. 要求代码优雅,分层设计,单个文件不要大于1000行.
